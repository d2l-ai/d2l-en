From Principle to Practice

We will explain deep learning and machine learning from scratch, and you do not need such background knowledge. Our goal is to offer an interactive learning experiene with math, code, images, text, and discussions, so you can understand and use deep learning.

Mathematics + Images + Code

We offer an interactive learning experience with mathematics, images, code, text, and discussions, where concepts and techniques are illustrated and implemented with experiments on real data sets.

Each section is an executable Jupyter Notebook

You can modify the code and tune hyperparameters to get instant feedback to accumulate practical experiences in deep learning.

Active community support

You can discuss and learn with thousands of peers in the community through the link provided in each section.
