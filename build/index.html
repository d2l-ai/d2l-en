Equations + Figures + Codes

Not only do we combine text, formulas, and diagrams to clarify the models and algorithms that are commonly used in deep learning, but we also provide code to demonstrate how to implement them from scratch and use real data to provide an interactive learning experience.

Each section is an runable Jupyter notebook

You can freely modify the code and hyperparameters to get timely feedback to accumulate practical experience in deep learning.

Active community support

You can discuss and learn with thousands of small partners in the community through the last link in each chapter.
