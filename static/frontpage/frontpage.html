Dive into Deep Learning

An interactive deep learning book with code, math, and discussions, based on the NumPy interface.

The previous NDArray version is here

Announcements

  • [Dec, 2019] A preview version (v0.7) is released. All the code has been rewritten with the NumPy API. Check out new instructions to run this book on Amazon SageMaker and Google Colab.
  • [Oct, 2019] We have improved Chapter: Preliminaries, added Chapter: Recommender Systems and Appendix: Mathematics for Deep Learning. To keep track of the latest updates, please follow D2L's open-source project.
  • [Jun, 2019] The Chinese version is the No. 1 best seller of new books in "Computers and Internet" at the largest Chinese online bookstore.
  • [May, 2019] Slides, Jupyter notebooks, assignments, and videos of the Berkeley course can be found at the syllabus page.
